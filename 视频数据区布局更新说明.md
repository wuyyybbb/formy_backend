## 🎬 左侧：视频容器

### 视频特性

- ✅ **自动播放**：`autoPlay`
- ✅ **循环播放**：`loop`
- ✅ **静音**：`muted`（允许自动播放）
- ✅ **移动端支持**：`playsInline`
- ✅ **视频路径**：`/landing/guide-demo.mp4`

### 视觉样式

- **比例**：16:9（aspect-video）
- **边框**：深色边框 + 圆角
- **装饰**：青色边框（Hover 时增强）
- **阴影**：大阴影效果
- **背景**：深色背景

### CSS 特效

```tsx
<div className="relative group">
  {/* 视频容器 */}
  <div className="aspect-video rounded-lg border shadow-2xl">
    <video autoPlay loop muted playsInline>
      <source src="/landing/guide-demo.mp4" type="video/mp4" />
    </video>
  </div>
  
  {/* 装饰边框（Hover 效果）*/}
  <div className="border-2 border-primary/20 group-hover:border-primary/40">
  </div>
</div>
```

## 🎬 视频要求

### 视频文件

- **位置**：`frontend/public/landing/guide-demo.mp4`
- **格式**：MP4（H.264 编码）
- **建议比例**：16:9
- **建议分辨率**：1920x1080 或 1280x720
- **文件大小**：建议 < 10MB（网页加载）

### 如果视频不存在

浏览器会显示：
```
您的浏览器不支持视频播放
```

请确保视频文件在正确位置。

## 💻 技术实现

### HTML5 Video

```html
<video autoPlay loop muted playsInline>
  <source src="/landing/guide-demo.mp4" type="video/mp4" />
  您的浏览器不支持视频播放
</video>
```

**属性说明**：
- `autoPlay`：页面加载时自动播放
- `loop`：循环播放
- `muted`：静音（浏览器要求）
- `playsInline`：iOS 内联播放

### Grid 布局

```css
/* 桌面端：两列 */
.grid-cols-1 lg:grid-cols-2 {
  grid-template-columns: 1fr 1fr;
}

/* 移动端：单列 */
@media (max-width: 1024px) {
  grid-template-columns: 1fr;
}
```

## 🔧 自定义调整

### 调整视频大小

**当前**：`aspect-video`（16:9）

**更宽**：
```tsx
<div className="aspect-[21/9]"> {/* 更宽的比例 */}
```

**更窄**：
```tsx
<div className="aspect-[4/3]"> {/* 4:3 传统比例 */}
```

### 调整数字大小

**当前**：`text-6xl md:text-7xl`

**更大**：
```tsx
<div className="text-7xl md:text-8xl">
```

**更小**：
```tsx
<div className="text-5xl md:text-6xl">
```

### 调整间距

**当前**：`space-y-12`（垂直间距 48px）

**更紧密**：
```tsx
<div className="space-y-8"> {/* 32px */}
```

**更宽松**：
```tsx
<div className="space-y-16"> {/* 64px */}
```

### 调整左右比例

**当前**：50% / 50%

**视频更大**：
```tsx
<div className="grid lg:grid-cols-[3fr_2fr]"> {/* 60% / 40% */}
```

**数据更大**：
```tsx
<div className="grid lg:grid-cols-[2fr_3fr]"> {/* 40% / 60% */}
```

## 🎯 设计理念

### 为什么左右布局？

1. **视觉平衡**
   - 视频吸引眼球
   - 数据提供说明
   - 左右对称

2. **内容丰富**
   - 动态演示（视频）
   - 静态数据（指标）
   - 互补展示

3. **用户体验**
   - 视频自动播放，吸引注意
   - 数据就在旁边，方便阅读
   - 一眼看完所有信息
