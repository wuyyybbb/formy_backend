## 🎬 横向滚动动画

### 实现方式

1. **无限循环滚动**
   - 8 张服装图片自动从右向左滚动
   - 图片重复两次，实现无缝循环
   - 滚动速度：40 秒完成一轮

2. **交互效果**
   - 鼠标悬停：暂停滚动
   - 图片放大：Hover 时图片缩放 1.1 倍
   - 发光效果：Hover 时显示青色光晕

3. **边缘渐隐**
   - 左右两边有渐变遮罩
   - 创造无限延伸的视觉效果

## 🔧 自定义选项

### 调整滚动速度

在 `LandingPage.tsx` 中修改：

```css
.scrolling-content {
  animation: scroll-left 40s linear infinite;
  /* 改成 30s = 更快 */
  /* 改成 60s = 更慢 */
}
```

### 调整图片数量

在代码中修改数组：

```tsx
{[1, 2, 3, 4, 5, 6, 7, 8].map((num) => (
  // 可以增加或减少数字
))}
```

### 调整图片尺寸

修改 `w-64 h-80` 类：

```tsx
<div className="w-64 h-80 rounded-lg ...">
  {/* w-64 = 宽度 256px */}
  {/* h-80 = 高度 320px */}
</div>
```
