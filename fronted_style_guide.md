1. 色彩体系（Color System）
主色（Primary Color）
#111111

点缀色（Accent Color）
鲜橙色（强调按钮、小型提示）
#FF2401

背景色（Background Colors）

主背景：#111111

卡片背景：#151515

文字颜色（Text Colors）

主文字：#FFFFFF

次文字：#BBBBBB

弱提示文字：#777777

高亮文字：使用主色 #00E2C7

边框与线条（Border Colors）

默认边框：#2B2B2B

高亮边框：#00E2C7

2. 字体系统（Typography）
字体

统一使用：Inter / Helvetica Neue / 默认无衬线字体

标题层级

H1：大号标题，48–64px，粗体

H2：区块标题，32–40px，半粗体

H3：小标题，24–28px

正文字号

正文大号：18px

正文默认：16px

辅助说明：14px

行高建议：1.6

整体要求风格：
高对比、大标题、科技感、清晰易读。

3. 组件圆角与阴影（Radius & Shadow）
圆角

小圆角：4px

默认圆角：8px

卡片圆角：12px

阴影

卡片阴影柔和，偏下方扩散，深度中等，用于“科技感但不浮夸”

4. 布局体系（Layout System）
最大内容宽度

主体内容宽度最大为：1200px

间距体系

超小间距：4px

小间距：8px

中等间距：12px / 16px

大间距：24px

超大区块间距：32px / 48px

整体要求：
大留白、强呼吸感、模块之间明显分区。

5. 背景视觉（Background Style）

Formy 视觉参考 Silana：

背景特征

深黑背景 + 细密点阵（Tech Grid）

点阵颜色为主色的低透明度版本

用于：首页、产品展示、空状态、Hero 区域

点阵风格关键点

点大小：1–2px

点间距：8–12px

点颜色：鲜橙色低透明（弱光）

整体效果：科技感、稳定、低噪音

6. 按钮风格（Button Style）
主按钮（Primary）

主色背景：#00E2C7

文字黑色

大号、圆角适中（8px）

风格：大胆、清晰、突出

次按钮（Secondary）

背景透明

鲜橙色描边

悬停时出现轻微鲜橙色光晕或浅背景

按钮形态要求：
整洁、科技感、小动效。

7. 卡片风格（Cards）

深背景：#151515

轻微透明边框（偏白或偏鲜橙色）

中等圆角

大内边距

悬停微亮 / 边框变青色

效果：低调但高级

8. 输入框风格（Input Fields）

深背景

深灰色边框

聚焦时边框变成主色青绿色

文字清晰、对比度足够

适合暗色 UI 的简洁科技感风格

9. 响应式规范（Responsive）
断点

手机 <768px

平板 768–1024px

桌面 ≥1024px

布局切换规则

桌面：左右结构（左参数、右预览）

手机：上下结构（上预览、下操作）

按钮在手机端固定底部，提高可点性

10. 整体风格总结（Overall Style Summary）

Formy 的视觉风格定位：

深色科技感（Dark-Tech）

青色主色（未来科技）

点阵视觉（Data Grid）

大标题、强对比、干净排版

强调专业、质量、稳定感

适合 AI 服装、摄影从业者使用的专业工具风格

一句话总结：

“深黑 + 鲜橙色霓虹 + 点阵科技感 + 清晰专业排版”

11. 使用建议

保持整体风格统一：#111111为唯一主色

点缀橙色只用在少量突出信息

多用留白，避免拥挤

不使用大面积渐变，保持简洁

所有页面保持黑色基调一致

字体始终使用一致的无衬线系统风格
