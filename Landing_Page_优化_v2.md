## 📐 技术细节

### 图片尺寸计算

原始图片：840 x 1256 像素

显示尺寸：
- 宽度：210px（840 ÷ 4）
- 高度：314px（1256 ÷ 4）

这样可以确保：
1. 保持原始比例
2. 适配不同屏幕
3. 滚动流畅

### CSS 实现

```css
background-color: var(--card-bg);
```


## 🔍 如果需要进一步调整

### 调整图片大小

在代码中修改 `style` 属性：
```tsx
<div style={{ width: '210px', height: '314px' }}>
  // 可以改成：
  // width: '280px', height: '419px'（1:3 比例）
  // width: '168px', height: '251px'（1:5 比例）
</div>
```

### 调整标题大小

修改 className：
```tsx
<h1 className="text-5xl md:text-7xl lg:text-8xl">
  // 可以改成：
  // text-4xl md:text-6xl lg:text-7xl（更小）
  // text-6xl md:text-8xl lg:text-9xl（更大）
</h1>
```

### 调整副标题

直接修改文字内容：
```tsx
<p>专为服装人而生的商用级 AI</p>
```
