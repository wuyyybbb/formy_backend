### 1. 整体布局
- 响应式网格布局（4列）
- 移动端自适应为单列显示
- 深色背景 `bg-dark`

### 2. 4 个价格套餐

#### STARTER（入门版）
- 价格：¥49/月（原价 ¥59）
- 算力：2000/月
- 约 50 张图片生成
- 基础功能

#### BASIC（基础版）
- 价格：¥99/月（原价 ¥119）
- 算力：5000/月
- 约 120 张图片生成
- 全部 AI 功能 + 批量处理

#### PRO（专业版）- **推荐**
- 价格：¥199/月（原价 ¥249）
- 算力：12000/月
- 约 300 张图片生成
- 极速处理 + 优先客服
- 特殊标记：顶部有"推荐"徽章
- 高亮边框：`border-primary/50`
- 按钮样式：`btn-primary`（其他为 `btn-secondary`）

#### ULTIMATE（旗舰版）
- 价格：¥399/月（原价 ¥499）
- 算力：30000/月
- 约 750 张图片生成
- 最高优先级 + 专属客服 + API 访问

## 🎯 核心元素

### 每个卡片包含：

1. **套餐名称**（大写英文）
   ```tsx
   <h3>STARTER / BASIC / PRO / ULTIMATE</h3>
   ```

2. **价格展示**
   - 大号价格：`text-5xl font-bold text-primary`
   - 人民币符号：`¥`
   - 节省金额：橙色 `text-accent`
   - 原价：删除线 `line-through`

3. **算力说明**
   - 每月算力额度
   - 数字用 primary 颜色高亮

4. **行动按钮**
   - PRO 套餐：`btn-primary`（青色高亮）
   - 其他套餐：`btn-secondary`（灰色边框）

5. **功能列表**
   - 使用勾选图标（SVG checkmark）
   - 青色图标 + 灰色文字
   - 响应式间距

## 🔧 技术实现

### 响应式布局
```tsx
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-6">
```

- 移动端：1 列
- 平板：2 列
- 桌面：4 列

### 卡片样式
```tsx
className="card group hover:border-primary/50 transition-all duration-300"
```

- 使用全局 `card` 类
- 悬停时边框变亮
- 平滑过渡动画

### PRO 套餐推荐徽章
```tsx
<div className="absolute -top-3 left-1/2 transform -translate-x-1/2">
  <span className="bg-primary text-dark text-xs font-bold px-4 py-1 rounded-full">
    推荐
  </span>
</div>
```

### 图标（SVG Checkmark）
```tsx
<svg className="w-4 h-4 text-primary mt-0.5 flex-shrink-0" fill="none" viewBox="0 0 24 24" stroke="currentColor">
  <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M5 13l4 4L19 7" />
</svg>
```

## 🎯 视觉效果

1. **深色主题**：与整体页面保持一致
2. **青色强调**：价格数字和图标使用 `text-primary`（青色）
3. **橙色提示**：节省金额使用 `text-accent`（橙色）
4. **卡片悬停**：边框变亮，提供交互反馈
5. **清晰层次**：
   - 标题 → 价格 → 算力 → 按钮 → 功能列表
   - 使用边框分隔不同区域
